# Omok Arena 🎮

**Figma 디자인 기반의 모던한 오목 게임**

Flutter로 개발된 전략적 오목 게임으로, 다크 테마와 고급스러운 UI/UX를 자랑합니다.

## 🎯 게임 개요

Omok Arena는 전통적인 오목 게임을 현대적인 디자인으로 재해석한 크로스 플랫폼 게임입니다. Figma에서 디자인된 컴포넌트를 기반으로 구축되어 일관성 있고 세련된 사용자 경험을 제공합니다.

### ✨ 주요 특징

#### 🎨 **디자인 시스템**
- **Figma 기반 UI**: 전문적으로 디자인된 컴포넌트 시스템
- **다크 테마**: 통합된 블랙 배경 (#000000)
- **Cafe24Ohsquare 폰트**: 한국어에 최적화된 타이포그래피
- **그라데이션 버튼**: 색상별 기능 구분 (파란색/주황색/빨간색/노란색)
- **캐릭터 마스코트**: 호랑이🐯와 토끼🐰 캐릭터

#### 🎮 **게임 기능**
- **13x13, 17x17, 21x21 보드**: 다양한 보드 크기 지원
- **AI 연습게임**: 혼자서도 즐길 수 있는 AI 대전
- **2인 플레이**: 로컬 멀티플레이어 지원
- **실시간 타이머**: 30초 애니메이션 카운트다운
- **캐릭터 시스템**: 12지신 캐릭터와 고유 스킬
- **아이템 시스템**: 전략적 게임 플레이를 위한 아이템
- **복권 시스템**: 코인과 아이템을 얻을 수 있는 복권

#### 🔧 **기술적 특징**
- **상태 관리**: Provider 패턴으로 효율적인 상태 관리
- **애니메이션**: 부드러운 전환과 시각적 피드백
- **오디오**: 게임 사운드와 효과음
- **반응형 디자인**: 다양한 화면 크기 지원

## 🎨 디자인 시스템

### 색상 팔레트
- **배경**: `#000000` (블랙)
- **주 버튼**: `#2196F3` (파란색) - AI 연습게임, 주요 액션
- **부 버튼**: `#FF9800` (주황색) - 2인 플레이
- **강조**: `#F44336` (빨간색) - 온라인 플레이, 경고
- **보조**: `#FFC107` (노란색) - 규칙안내, 구입/상점
- **성공**: `#4CAF50` (초록색) - 회원가입, 시작 버튼
- **중성**: `#757575` (회색) - 설정, 중립 액션

### 타이포그래피
- **제목**: `Cafe24Ohsquare` (32px, letter-spacing: -0.66)
- **버튼**: `Cafe24Ohsquare` (20px, letter-spacing: -0.30)
- **본문**: `Cafe24Ohsquare` (16px, letter-spacing: -0.20)

### 컴포넌트
- **버튼**: 6px 흰색 테두리, 24px 둥근 모서리, 그라데이션 배경
- **카드**: 15px 둥근 모서리, 그림자 효과
- **입력 필드**: 흰색 배경 (투명도), 테두리 강조

## 🎮 게임 특징

### 오목 규칙
1. **흑돌 먼저**: 검은 돌이 먼저 시작
2. **5목 승리**: 가로/세로/대각선으로 5개 연결
3. **30초 제한**: 각 턴마다 30초 시간 제한
4. **렌주 규칙**: 고급 플레이어를 위한 금수 규칙

### 캐릭터 시스템
- **12지신 테마**: 각 캐릭터별 고유한 디자인과 스킬
- **스킬 시스템**: 게임당 1회 사용 가능한 특수 능력
- **아이템 연동**: 캐릭터와 아이템의 전략적 조합

### 아이템 시스템
- **게임당 2회**: 제한된 사용으로 전략성 증대
- **다양한 효과**: 공격/방어/특수 효과 아이템
- **복권 획득**: 코인으로 구매하거나 복권으로 획득

## 🎮 게임 플레이 가이드

### 시작하기
