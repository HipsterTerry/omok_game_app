# 🎮 Omok Arena - 귀여운 오목 게임

Flutter로 제작된 키치하고 귀여운 스타일의 오목 게임 앱입니다.

## ✨ 최신 업데이트 (v2.0)

### 🎨 캐릭터 이미지 통합
- **12간지 동물 캐릭터 PNG 이미지 적용**
  - 모든 캐릭터 아이콘을 실제 이미지로 교체
  - 캐릭터 선택 화면 상단에 애니메이션 효과와 함께 표시
  - 이미지 로딩 실패시 이모지로 자동 대체

### 🏠 홈화면 UI 대폭 개선
- **캐릭터 로고와 타이틀 통합**
  - 호랑이+토끼 캐릭터 이미지와 "Omok Arena" 타이틀을 Stack으로 겹치게 배치
  - 캐릭터의 바둑돌이 타이틀 'k'에 거의 닿을 정도로 밀착 배치
  - 타이틀 폰트 크기 대폭 확대 (42px/52px)

- **레이아웃 최적화**
  - 모든 UI 요소를 위로 올려 캐릭터와 타이틀에 밀착
  - 간격 최소화로 더욱 컴팩트한 디자인
  - 버전 정보는 하단에 충분히 띄워서 가독성 확보

### 🎯 캐릭터 선택 화면 개선
- **상단 4개 캐릭터 이미지 표시**
  - 쥐, 양, 용, 토끼 순서로 Bounce 애니메이션 적용
  - 호랑이 → 양으로 교체하여 더 다양한 캐릭터 노출
  - 48x48 크기로 최적화된 이미지 표시

- **캐릭터 카드 이미지 적용**
  - 모든 캐릭터 카드에서 아이콘 대신 PNG 이미지 사용
  - 선택시 흰색 블렌드 효과로 시각적 피드백 강화
  - 32x32 크기로 카드 내 최적 배치

### ⏰ 타이머 및 애니메이션 강화
- **30초 타이머 경고 효과**
  - 10초 이하 남으면 빨간색 깜빡임 효과
  - 다중 그림자와 글로우 효과로 시각적 경고 강화
  - AnimationController를 통한 부드러운 색상 전환

- **게임 시작 카운트다운 단순화**
  - "3, 2, 1, Start"만 큰 글씨로 표시
  - 불필요한 장식 제거로 집중도 향상
  - Cafe24Ohsquare 폰트로 일관성 유지

### 🎨 버튼 및 다이얼로그 스타일링
- **스킬 버튼 개선**
  - 정보 버튼: 그라데이션, 강화된 테두리, 다중 그림자
  - 사용 버튼: 텍스트 제거, 아이콘 크기 확대, 티어별 글로우
  - 스킬 상세 다이얼로그: 어두운 배경, 탭하면 닫힘 기능

- **홈화면 상단 버튼 통일**
  - 모든 버튼을 90x75 크기로 통일
  - 강화된 테두리와 그림자 효과
  - FittedBox로 오버플로우 방지

## 🎯 주요 기능

### 🎮 게임 모드
- **온라인 플레이** (개발 중)
- **2인 플레이 (로컬)** - 같은 기기에서 두 명이 플레이
- **1인 플레이 (AI 대전)** - AI와 대전

### 🐲 12간지 캐릭터 시스템
- **천급 캐릭터** (15% 승률 보너스)
  - 청룡 (용) - 천둥번개: 상대방의 다음 수 무효화
  - 백호 (호랑이) - 맹호의 일격: 상대방의 돌 하나 제거
  - 현무뱀 (뱀) - 독사의 독: 상대방 시간 30초 감소

- **지급 캐릭터** (8-10% 승률 보너스)
  - 천리마 (말) - 질주: 연속으로 두 번 둘 수 있음
  - 영리원숭이 (원숭이) - 속임수: 돌 위치 이동
  - 금계 (닭) - 새벽울음: 시간 30초 추가
  - 충견 (개) - 수호: 상대방 공격 스킬 무효화

- **인급 캐릭터** (2-4% 승률 보너스)
  - 쥐돌이 (쥐) - 재빠른 움직임: 상대방 마지막 수 확인
  - 황소 (소) - 완고함: 모든 효과 무시
  - 행운토끼 (토끼) - 행운: 랜덤 좋은 효과
  - 평화양 (양) - 평화: 양 플레이어 시간 30초씩 추가
  - 복돼지 (돼지) - 황금돼지: 게임 종료 후 코인 2배

### ⚡ 스킬 시스템
- **공격형**: 상대방에게 직접적인 피해
- **방어형**: 자신을 보호하거나 강화
- **교란형**: 게임 상황을 혼란시킴
- **시간제어형**: 게임 시간을 조작

### 🎲 캐릭터 뽑기 시스템
- **동/은/금 복권** 시스템
- 캐릭터 조각, 아이템, 코인, 경험치 획득
- 확률형 보상 시스템

## 📁 프로젝트 구조
```

├── screens/          # 화면들
├── widgets/          # 커스텀 위젯들
├── models/           # 데이터 모델들
├── services/         # 비즈니스 로직
```
